LA PROPUESTA

Diseñá la landing page
de tu consultora

Cada estudiante elige una consultora de Marketing, RRHH o comunicación — real o imaginaria — y crea una landing page completa usando IA como herramienta de producción.

Brief

Definís el servicio, el público y el tono. Esto guía a la IA para generar contenido coherente.

Copy con IA

Usás una herramienta de IA para generar la estructura completa: headlines, secciones y CTAs.

3 estilos visuales

Aplicás tres identidades visuales distintas al mismo contenido. Cada versión tiene su propia personalidad.

CÓMO HACERLO

El proceso, paso a paso

Seguí este flujo de trabajo para construir tu landing de principio a fin.

01

Elegí tu consultora

Puede ser real o ficticia. Definí el nombre, el servicio principal y el público al que apunta.

02

Escribí el brief

Redactá un documento breve con: qué ofrece, a quién le habla, cuál es el tono y qué acción querés que tome el visitante.

03

Generá con IA

Compartí el brief con una herramienta de IA y pedile que genere la landing completa. Editá y ajustá el resultado.

04

Publicá las 3 versiones

Subí los tres archivos HTML a la plataforma de hosting que prefieras y compartí las URLs.

QUÉ ENTREGAN

4 entregables

Todo debe estar publicado y accesible desde una URL. No se aceptan archivos locales ni capturas de pantalla.

✓

Brief

El documento estratégico de tu consultora. Puede ser un Google Doc o un archivo de texto.

✓

Landing principal

La versión base en HTML con el copy completo. Publicada y accesible desde una URL.

✓

3 versiones visuales

Tres archivos HTML con el mismo contenido pero estilos visuales distintos. Las tres publicadas.

✓

Reflexión

Un párrafo corto (150–200 palabras) sobre el proceso: qué decidiste, qué cambiarías y qué aprendiste.
